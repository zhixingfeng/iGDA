Class RankDictionary
A rank dictionary is a data structure storing the rank of an element in a sequence at every position of the sequence. Only supports sequences containing SimpleType or bool.

All Subcl's Levels, WaveletTree
Defined in <seqan/index.h>
Signature template <typename TValue, typename TSpec> struct RankDictionary;
